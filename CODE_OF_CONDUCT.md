# Code of Conduct

This project follows the **Contributor Covenant v2.1**.

The full text is maintained canonically at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — opening issues, submitting pull requests, commenting on discussions — you agree to abide by its terms.

## Reporting

To report a concern, please contact the maintainers via the repository's
**Security advisories** tab on GitHub, or email the maintainers listed in the repository settings.

Reports will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of reporters.
